# 3D_Models

STL-файлы системы **Breadboard Maketka**. Имена должны совпадать с теми, на
которые ссылается `index.html` (атрибут `data-stl`). Геометрия каждой детали
описана в `src/brackets.js` — это единый источник правды (форма, габарит,
отверстия на сетке 15 мм).

Все актуальные STL лежат в папке `STL` и перечислены ниже.

## Макеты (коннекторы)

| Макет     | Основной файл | Боковушка (шина, печатать ×2) |
|-----------|---------------|-------------------------------|
| Маленький | `s.STL`       | —                             |
| Средний   | `m.STL`       | `m_k.STL`                     |
| Большой   | `l.STL`       | `l_k.STL`                     |

## Перемычки (крепёж, сетка 15 мм)

Плоские накладки, ложатся на стык плат и крепятся в отверстия сетки.
Имена кодируют ширину и длину (в миллиметрах) согласно названию.

| Тип            | Файлы                                                                                          |
|----------------|--------------------------------------------------------------------------------------------------|
| **I** w·15     | `br_15_30.STL` `br_15_45.STL` `br_15_60.STL` `br_15_90.STL` `br_15_180.STL`                      |
| **I** w·30     | `br_30.STL` `br_30_30.STL` `br_30_45.STL` `br_30_60.STL` `br_30_90.STL` `br_30_180.STL`          |
| **I** w·60     | `br_60_45.STL` `br_60_60.STL`                                                                    |
| **I** w·90     | `br_90_45.STL` `br_90_60.STL`                                                                    |
| **I** w·180    | `br_180_60.STL` `br_180_90.STL` `br_180_180.STL`                                                  |
| **L** угол 90° | `br_L_45.STL` `br_L_60.STL`                                                                      |
| **Т** тройник  | `br_t_90_45.STL`                                                                                |
| **✕** крест    | `br_x_90_90.STL`                                                                                |

## Дополнительные уголки

|--------------|-------------------------------------|
